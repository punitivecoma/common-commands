Common Command Tasks

Repair Commands

The sequence of running DISM /Online /Cleanup-Image /RestoreHealth followed by sfc /scannow creates a robust strategy for repairing Windows installations. It first fixes potential damage at the source before verifying and restoring the integrity of essential system files. This approach maximizes the chances of resolving errors and enhances system stability.

DISM Command

DISM /Online /Cleanup-Image /RestoreHealth

SFC Command

sfc /scannow

Networking Commands

The command prompt commands related to networking are powerful tools for managing, testing, and troubleshooting network configurations. Here’s a breakdown of some common network commands in Windows, along with their purposes:

1. IP Configuration

Purpose: Displays the current IP configuration for all network interfaces.

Usage: ipconfig /all shows detailed information including the IP address, subnet mask, and default gateway.

ipconfig /all

2. Flush DNS

Purpose: Clears the DNS resolver cache to resolve DNS-related issues.

Usage: ipconfig /flushdns clears the DNS cache, which can help fix problems accessing websites.

ipconfig /flushdns

3. Ping Command

Purpose: Tests connectivity between your computer and another network device.

Usage: ping [hostname/IP] sends packets to the specified address to check if it’s reachable.

ping google.com

4. Tracert Command

Purpose: Traces the route packets take to a specified destination.

Usage: tracert [hostname/IP] shows each hop along the route to the target.

tracert google.com

5. Netstat Command

Purpose: Displays active network connections and listening ports.

Usage: netstat -ano shows connections with associated Process IDs (PIDs).

netstat -a

6. Nslookup Command

Purpose: Queries DNS to obtain the domain name or IP address mapping.

Usage: nslookup [domain name] retrieves the IP address for a specified domain.

nslookup example.com

7. Route Command

Purpose: Displays and modifies the IP routing table.

Usage: route print shows current network routes; route add [destination] [mask] [gateway] creates a new route.

route print

8. ARP Command

Purpose: Displays and modifies the ARP (Address Resolution Protocol) cache.

Usage: arp -a shows the current ARP entries.

arp -a

9. Netsh Command

Purpose: A versatile command for managing network configurations.

Usage: netsh interface show interface displays the status of network interfaces.

netsh interface show interface

Troubleshooting Windows Update Errors

Use the following commands to troubleshoot issues related to Windows Updates. These commands stop Windows Update services, delete temporary files, and restore system files to ensure a smooth update process.

1. Stop Windows Update Services

net stop wuauserv

2. Stop Cryptographic Services

net stop cryptSvc

3. Stop Background Intelligent Transfer Service

net stop bits

4. Stop Windows Installer Service

net stop msiserver

5. Delete SoftwareDistribution Folder

del %windir%\SoftwareDistribution /s /q

6. Start Windows Update Services

net start wuauserv

7. Start Cryptographic Services

net start cryptSvc

8. Start Background Intelligent Transfer Service

net start bits

9. Start Windows Installer Service

net start msiserver

10. Repair Windows Image

DISM /Online /Cleanup-Image /RestoreHealth

11. Verify System Files

sfc /scannow

Built-in Windows Tools

Windows provides a variety of built-in tools for system management, troubleshooting, and configuration. Use the following commands to access these useful utilities:

System Information

Command: systeminfo

Description: Displays detailed system information.

IP Configuration

Command: ipconfig

Description: Displays the current IP configuration for all network interfaces.

List Running Processes

Command: tasklist

Description: Lists all currently running processes.

User Accounts

Command: net user

Description: Displays all user accounts on the system.

Disk Usage

Command: wmic logicaldisk get size,freespace,caption

Description: Shows used and free disk space on all drives.

Current Active Sessions

Command: query user

Description: Lists user sessions on the local machine.

Device Manager

Command: devmgmt.msc

Description: Opens the Device Manager.

Task Manager

Command: taskmgr

Description: Opens the Task Manager.

Event Viewer

Command: eventvwr

Description: Opens the Event Viewer.

Control Panel

Command: control

Description: Opens the Control Panel.

System Configuration

Command: msconfig

Description: Opens the System Configuration tool.

Performance Monitor

Command: perfmon

Description: Opens the Performance Monitor.

Windows Update

Command: wuapp

Description: Opens Windows Update settings.

Registry Editor

Command: regedit

Description: Opens the Windows Registry Editor.

Disk Cleanup

Command: cleanmgr

Description: Opens the Disk Cleanup tool.

Windows Security

Command: windowsdefender:

Description: Opens Windows Security settings.

Group Policy Editor (Pro editions)

Command: gpedit.msc

Description: Opens the Group Policy Editor.

Services

Command: services.msc

Description: Opens the Services management console.

Windows Memory Diagnostic

Command: mdsched

Description: Opens the Windows Memory Diagnostic tool.

Resource Monitor

Command: resmon

Description: Opens Resource Monitor for monitoring system performance.

Command Prompt as Administrator

Command: cmd followed by runas /user:Administrator cmd

Description: Opens Command Prompt with administrative privileges.

System Information Utility

Command: msinfo32

Description: Displays comprehensive hardware and software details.

Network Connections

Command: ncpa.cpl

Description: Opens Network Connections control panel.

Windows Features

Command: optionalfeatures

Description: Opens the Windows Features dialog.

DirectX Diagnostic Tool

Command: dxdiag

Description: Opens the DirectX Diagnostic Tool.
